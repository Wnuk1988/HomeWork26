Phoenix is a superhero!

Джин Грей – одна из участников первого состава команды Людей Икс. Важный персонаж для Профессора Икс, он для нее словно приемный отец, наставник.

Росомаха – ее надежный и верный друг.

Суперсила:

  1. Телекинез, телепатия;
  2. Умеет летать;
  3. Контроль материи на атомном уровне;
  4. Контроль над жизненной энергией, самовоскрешение;
  5. Создание космической энергии;
  6. Изменение пространства и времени, реальности.

Недостатки:

  1. Не всегда справляется с собственной мощью, может перейти на темную сторону;
  2. Спонтанные потери памяти.
